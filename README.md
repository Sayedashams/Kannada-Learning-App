# Kannada Learning App – UI/UX Design

## Project Overview

Designed a responsive UI/UX prototype for a Kannada Learning App aimed at helping users learn Kannada through an intuitive and engaging interface. The project includes both mobile and web designs created in Figma.

---

## Features

- Responsive Mobile UI
- Responsive Web UI
- User Authentication Screens
- Course Dashboard
- Interactive Learning Flow
- Modern and Clean Interface
- User-Centered Design

---

## Technologies & Tools

- Figma
- UI/UX Design
- Wireframing
- Prototyping
- User Flow Design
